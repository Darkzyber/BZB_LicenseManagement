# วันที่ Task Phase ผู้รับผิดชอบ สถานะ
1 19–20 มีนา Update 4 SharePoint Lists ให้ตรง column spec — SoftwareCatalog, Licenses, LicenseHistory, ApprovalRequests SharePoint IT Admin / HR Admin วันนี้ ⬅
2 20 มีนา เพิ่ม Sample Data ใน SoftwareCatalog + Licenses — ทดสอบ Lookup + Calculated column DaysUntilExpiry SharePoint IT Admin รอ
3 23 มีนา Flow 1 — Expiry Notification: Recurrence trigger · DaysLeft expression · Send email 3 tier · Update LicenseStatus Flow IT Admin รอ
4 24 มีนา Flow 2 — Approval Renew: When item created in ApprovalRequests · Start and wait for approval · Approved/Rejected branches · LicenseHistory Flow IT Admin รอ
5 25 มีนา Flow 3 — Resign/Transfer/Convert: PowerApps v2 trigger · Switch ActionType · LicenseHistory ทุก license · Email summary IT+HR Flow IT Admin รอ
6 26 มีนา Flow 4+5 — New Flows (ใหม่): Request Notification → Approval · Route Selection → แจ้ง Procurement หรือ Finance แล้วแต่ Route New ✨ IT Admin รอ
7 27, 30 มีนา Power Apps — Dashboard + All Licenses: KPI Cards · Gallery filter/search · ปุ่ม Renew/Revoke · History Timeline Power Apps IT Admin รอ
8 30 มีนา Power Apps — New Request Form (อัปเดต): Payment Route Dropdown · Dynamic fields · เรียก Flow 4+5 · Pending Purchase status New ✨ IT Admin รอ
9 30 มีนา Set Permissions: สร้าง SharePoint Groups (HR Admin, IT Admin, Manager, Employee) · Item-level permissions · Power Apps role check (isHRAdmin) Perm HR Admin รอ
10 31 มีนา ⚑ End-to-End Testing + Deploy: ทดสอบ 10 Test Cases · แก้ Bug · Share Power App ให้ HR team · UAT กับผู้ใช้จริง Testing HR Admin + IT Deadline
